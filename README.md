Geo I/O WKT
===========

[![Build Status](https://travis-ci.org/geo-io/wkt.png?branch=master)](https://travis-ci.org/geo-io/wkt)

Geo I/O WKT (well-known text) package.

Installation
------------

Install [through composer](http://getcomposer.org).

```json
{
    "require": {
        "geo-io/wkt": "0.1.*@dev"
    }
}
```
